#  GW Detector & ML Framework

Проект посвящен моделированию отклика гравитационно-волнового детектора и применению машинного обучения (1D-CNN) для детектирования сигналов от слияния черных дыр и искусственных источников.

##  О проекте

Данный фреймворк позволяет:
1.  **Симулировать** отклик интерферометра (типа LIGO) на гравитационные волны.
2.  **Генерировать** обучающие выборки с физически корректным шумом и мульти-детекторной корреляцией.
3.  **Обучать** нейронную сеть для классификации сигналов (Signal vs Noise).
4.  **Валидировать** модель на реальных астрофизических данных из каталога SXS.

##  Стек технологий

- **Языки:** Python 3.8+, C++17
- **Библиотеки:** PyTorch, NumPy, Pandas, Matplotlib, Scikit-learn
- **Данные:** SXS Gravitational Waveform Database
